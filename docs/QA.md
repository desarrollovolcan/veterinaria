# QA Manual - Plataforma Veterinaria

## Validación por módulo (patrón obligatorio)
1. Entrar desde menú al módulo.
2. Verificar que exista **Formulario arriba**.
3. Verificar que exista **Tabla abajo**.
4. Probar crear, editar y eliminar/inactivar con confirmación.
5. Probar filtros y paginación.

## Módulos cubiertos
- Propietarios, Mascotas, Veterinarios, Citas, Ficha Clínica, Vacunas, Desparasitación
- Inventario/Farmacia, Facturación/Caja, Hospitalización, Cirugías, Laboratorio, Reportes
- RBAC/Accesos, Parametrización, Catálogos Maestros, Servicios/Tarifario
- Proveedores/Compras, Morosos/CxC, Auditoría/Bitácora
- Documentos/Consentimientos, Comunicaciones/Recordatorios, Portal cliente/Reserva online
